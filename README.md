HOSTEL MANAGEMENT SYSTEM

Project Description:
This project is a Hostel Management System developed in C++ using Object-Oriented Programming concepts. The system allows students to reserve hostel beds, manage student records, and store data using file handling.

Features:

- Student Registration
- Hostel Bed Reservation
- Student Record Management
- File Handling
- OOP Concepts Implementation

OOP Concepts Used:

- Inheritance
- Encapsulation
- Abstraction
- Static Polymorphism
- Dynamic Polymorphism

Files:

- hms2.cpp
- Hostel.txt
- Student.txt

How to Run:
Compile the source code using a C++ compiler and execute the generated program.
